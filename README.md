# BuildX Analytics — IPL Dataset (2008–2025)

## Dataset Selected
IPL Ball-by-Ball Dataset (2008–2025), sourced from Kaggle.

## Why I Selected It
I chose this dataset because IPL contains match details, player and data across 17 seasons. I would like to learn how team performance and player stats have evolved over time.

## Business Questions Answered
1. Which teams have scored the most total runs across all IPL seasons?
2. Who are the top individual run-scorers in IPL history?
3. Which bowlers have the best (lowest) economy rates among high-volume bowlers?

## Tools Used
- **Python (Pandas)** — Data cleaning and preparation
- **SQL (SQLite)** — Business analysis queries
- **Power BI** — Interactive dashboard and visualization
- **GitHub** — Project documentation and publishing

## Key Insight
Mumbai Indians have scored the highest cumulative runs across all seasons, totalling over 46,014 runs, reflecting their consistent batting strength since 2008
